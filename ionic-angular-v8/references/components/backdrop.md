## Properties

| Prop | Attribute | Type | Default | Description |
| --- | --- | --- | --- | --- |
| `stopPropagation` | `stop-propagation` | `boolean` | `true` | If `true`, the backdrop will stop propagation on tap. |
| `tappable` | `tappable` | `boolean` | `true` | If `true`, the backdrop will can be clicked and will emit the `ionBackdropTap` event. |
| `visible` | `visible` | `boolean` | `true` | If `true`, the backdrop will be visible. |
